# Inventory Management System

A Python-based desktop application built using Tkinter to manage inventory records efficiently.

## Features

* Add, delete, and search products
* Real-time stock management
* Calculate total inventory value
* Export inventory data to CSV
* Simple graphical user interface

## Technologies Used

* Python
* Tkinter
* CSV File Handling

## Purpose

Designed to simplify inventory tracking for small shops or warehouses.
